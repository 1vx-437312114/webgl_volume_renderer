webgl 2.0 demo - Volume Rendering by 1vx 437312114
using the new 3D texture and GLSL 3.0 es shader features
Shader code based on the technique described in
Acceleration Techniques for GPU-based Volume Rendering
(J. Krüger, R. Westermann, IEEE Visualization 2003).
Volume taken from the Volvis Library (Gris Uni-Tübingen)
Spin the volume by dragging it with the mouse, or use cursor-keys (up-/down -> x-axis, left-/right -> y-axis)
